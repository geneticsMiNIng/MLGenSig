Function expr_nbinom

Function expr_nbinom tests for differences between the base means of two conditions. Beside a data set containing read counts, we need a description of the samples, which we keep in a factor, whose elements correspond to different gorups.

In our example we will test for differential expression between groups with LumA breast cancer subtype and other subtypes of that cancer. We will use factor conditions, which consist of two values corresponds to subtype of breast cancer: LumA and other.

condition<-ifelse(BRCA_mRNAseq_chr17$SUBTYPE=="LumA","LumA","other")
head(condition,8)
## [1] "other" "LumA"  "LumA"  "LumA"  "LumA"  "LumA"  "other" "LumA"

Evaluation of expr_nbinom may take a few minutes

test.expr <- expr_nbinom(BRCA_mRNAseq_chr17[,-1], condition)

As a result we obtain a data frame contains among others: base mean, means for each group and p-value of test.
